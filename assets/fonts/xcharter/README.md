# XCharter

Self-hosted WOFF2 for body typography. Based on Bitstream Charter (Michael Sharpe, SIL Open Font License).

Source: [CTAN xcharter](https://ctan.org/pkg/xcharter)

Regenerate WOFF2 from OTF (optional): `brew install woff2` then `../utils/convert-fonts-to-woff2.sh` with source OTF files in this directory.
